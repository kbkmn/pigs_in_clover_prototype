# pigs_in_clover_prototype

Прототип игры для смарт-часов: лабиринт с металлическим шариком. Один уровень в виде буквы П, деревянный. Колесо крутит лабиринт вокруг шарика, гравитация всегда вниз, камера следует за шариком. Выход — дырка в полу правой ноги. Есть стрелка-указатель на выход и миникарта сверху.

## Запуск

```
npm install
npm run dev
```

## Управление в браузере

- тап / пробел — действие
- скролл мыши, ↑/↓, перетаскивание колеса — вращение
