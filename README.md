# pigs_in_clover_prototype

Прототип игры для смарт-часов. Скафолд: корпус часов, экран-canvas, колесо (тап + вращение), измеритель об/с. Игры пока нет.

## Запуск

```
npm install
npm run dev
```

## Управление в браузере

- тап / пробел — действие
- скролл мыши, ↑/↓, перетаскивание колеса — вращение
